This site was made to make it easier to find the lecture notes and coursework files for our modules, and ultimately to be able to keep track of it all.

URLs

We decided that the current system needed tidying up and we really wanted nice hackable URLs; no more of these "What's the lecturer's middle initial?" problems when you just want a module.

To make this even easier, if you try a 'logical' url then we should be able to use it to guess what you want. There's also a short-url system to provide shortcuts to specific modules or lectures.

People

We are Ben (Designer), Rob (Django), Rob (Django & Server Admin) & Marcus (Javascript). All of us first-year undergraduates in the Computer Science Dept.

Technology

The whole thing is built in Django (the web framework for ponies with magical powers) and hosted on Ubuntu Server 9.04.

We use semantic XHTML & CSS that should even be compatible with IE, though you should all know better than to be using it..
